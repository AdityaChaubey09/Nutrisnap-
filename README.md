🥗 NutriSnap – AI Nutrition Lens

NutriSnap is an AI-powered nutrition assistant that helps users make healthier food choices in a fun and engaging way. By simply taking a photo of your meal, NutriSnap analyzes the food, provides detailed nutritional information, assigns a Nutrition Score, and even suggests healthier alternatives.

🌟 Features

✅ Meal Recognition – Snap or upload a picture of your food, and NutriSnap automatically detects the meal.
✅ Nutrition Insights – Get detailed values like calories, proteins, fats, carbs, and more.
✅ Nutrition Score – Each meal is scored to help you understand how healthy it is.
✅ Healthier Alternatives – Suggestions for better food choices to improve diet quality.
✅ Meal History – Track your past meals and monitor progress over time.
✅ Gamification 🎮 – Earn Nutrition Points based on your daily eating habits.
✅ Interactive Dashboard – View meal history, points, and insights in one place.

🛠️ Tech Stack
Frontend

⚛️ React.js – Interactive and responsive user interface

🎨 Tailwind CSS – Modern styling for a clean design

Backend

⚡ Node.js + Express.js – Handles API requests and server logic

🔗 Integration with external APIs for food recognition and nutrition data

Database

🍃 MongoDB – Stores user profiles, meal records, and nutrition history

AI & APIs

🤖 Food Recognition Models – e.g., Clarifai / Hugging Face Food101

🥦 Nutrition Data API – e.g., Nutritionix for calories & nutrient breakdown

📂 Project Structure

NutriSnap/
├── client/        # React frontend
├── server/        # Node.js backend
├── models/        # MongoDB schemas
├── routes/        # API routes
└── README.md      # Documentation

🚀 How It Works

📸 User uploads/takes a picture of their meal

🤖 AI model detects the food items

🥦 Nutrition API fetches calorie & nutrient info

🎯 Nutrition Score is calculated

📊 Data stored in MongoDB for history tracking

🏆 Users earn Nutrition Points for healthy meals
